# soen-345
Academic Project, Winter 2026, SOEN 345: Software Testing, Verification and Quality Assurance 

## How to run
1. Run `git clone https://github.com/Extinctable/soen-345.git` on the chosen folder in your device
2. Install & Open Android Studios
3. On the Projects Tab, Select **Open** and select the file `build.gradle.kts` in the folder `TicketReservationApp`
